#productivity

Проект для оценки продуктивности пользователя

1) Вывод дня с максимальной/минимальной продуктивностью

2) Сохранение истории в JSON-файл(для начала)

3) Целевые показатели и сравнение с ними

4) Подсчет количества дней выше/ниже среднего

Дополнения которые не обязательны, но стоят внимания:
1) Ввод дополнительных параметров (настроение, часы сна)
2) Простую текстовую диаграмму (с помощью символов █ или *)
3) Рекомендации на основе результата
4) Добавление класса/ов, для организации кода
5) Сравнение с предыдущими неделями
